# PostureGuard (MVP) - Project Charter

## 1. 核心目标
利用闲置 Android 手机的前置摄像头，实时监测用户坐姿。当判定为“不良坐姿”持续一定时间后，触发语音提醒。

## 2. 技术栈 (Tech Stack)
- **语言**: Kotlin
- **UI**: Jetpack Compose (Material3)
- **视觉引擎**: MediaPipe Pose (Google)
- **相机**: CameraX
- **语音**: Android Native TTS
- **架构**: MVVM + Flow

## 3. 功能模块 (MVP)
1.  **相机预览**: 实时显示画面（镜像处理）。
2.  **姿态检测**: 识别鼻子、耳朵、肩膀等关键点。
3.  **坐姿分析**: 
    - 计算头部倾斜角度。
    - 计算肩膀水平度。
    - 状态防抖（避免误报）。
4.  **反馈机制**: 
    - 视觉提示（红/绿框）。
    - 语音播报（TTS）。

## 4. 关键约束
- **前台运行**: Android 系统限制后台摄像头访问，App 需保持屏幕常亮（可提供省电黑屏模式）。
- **隐私**: 所有计算在本地完成，不上传任何图像。
